# frontend

## 프로젝트 실행방법

```
$ npm create vite@latest
$ cd vite-project
$ npm install
$ npm install -D vite-plugin-pwa
$ npm run dev
```

일단 다 썼는데 파일 받으시면 그냥 npm install 하고 run dev 바로 하시면 됩니당

## 규칙

1. 폴더는 소문자로 써서 생성해주세요.
2. 페이지명은 파스칼로 써주세요 예시: HomePage. 
3. 컴포넌트는 용도별로 디렉토리 나눠주세요.
4. 탭사이즈는 4개로 해주세요.
